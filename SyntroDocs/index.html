SyntroCore

SyntroCore is the fundamental building block of Syntro. It consists of:

  • Libraries:
    • SyntroLib - a library used by all Syntro apps.
    • SyntroGUI - a library that provides common dialogues for windowed Syntro apps.
    • SyntroControlLib - a library that allows apps to embed SyntroControl functionality.
  • Syntro apps:
    • SyntroExec - an app that starts and manages other Syntro apps.
    • SyntroControl - Syntro's distirbuted message switch and directory manager.
    • SyntroLog - an app to collect and display log entries from all apps in a Syntro cloud.
    • SyntroStore - an app that can collect and store multicast service streams from a Syntro cloud.
    • SyntroCFS - and app that provides remote access to multicast service data stored by SyntroStore.
